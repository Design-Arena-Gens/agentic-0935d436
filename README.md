# What Is Science?

A compact Next.js site that explains what science is, how the scientific method works, and why evidence-based inquiry remains our most reliable path to understanding nature.

## Getting Started

```bash
npm install
npm run dev
```

Visit `http://localhost:3000`.

## Production Build

```bash
npm run build
npm start
```

## Deployment

This project is ready for Vercel. Run `vercel deploy --prod --yes --token $VERCEL_TOKEN --name agentic-0935d436`.
